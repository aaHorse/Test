本仓库有两个项目，第一个是图书管理系统，第二个是在线聊天

##############################################################

图书管理系统：

  前言总结：
  
  本系统的重点在于代码的规整性，开发的框架参考了Android的mvp开发模式，在本项目，同样将代码分为三层进行开发，第一层是UI,
  第二层是主持者（负责第一层和第三层之间的通信），第三层是实体类以及访问网络、访问数据库等逻辑。
  
  本系统的数据库位于服务器上面，所以这一个项目下载就可以运行，不受限制。
